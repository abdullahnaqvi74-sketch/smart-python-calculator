#  Python Calculator Project

##  Project Description
This is my first Python programming project — a simple command-line calculator.

The calculator supports multiple mathematical operations and is built to practice Python fundamentals.

##  Features
✔ Addition of multiple numbers  
✔ Subtraction  
✔ Multiplication  
✔ Division with error handling  
✔ Power operation  
✔ Modulus operation  
✔ Calculation history tracking  
✔ Infinite menu loop system  

##  Concepts Used
- Functions  
- Loops (while loop)  
- Conditional statements  
- Lists  
- Exception handling (try-except)  
- Input processing
